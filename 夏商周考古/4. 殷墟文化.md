# 四、殷墟文化
## 发掘史
1. 王懿荣，官员、金石学家，1899年生病之后家人买来“龙骨”，懂药理，亲自过目药材，发现有文字的龟甲。从此开始大量购买。
2. 1900八国联军事后，王懿荣投笔从戎，后来携夫人跳井自杀以身殉国，甲骨由儿子卖给刘鹗。刘鹗，小说家，代表作《老残游记》。刘鹗将1058板拓片出版著录书《铁云藏龟》。后来被发配充军。
3. 孙诒让，学者。根据《铁云藏龟》出版考释甲骨文著作《契文举例》。
4. 罗振玉，现代农学开拓者、考古学奠基人。当时甲骨的价格已经飞涨，商人对出土地保密。罗振玉调查到了出土地为安阳小屯村，并且定性为殷商王室遗物，洹水之滨。
5. 王国维，艺术、历史、古文字、历史学家。《殷卜辞中所见先公先王考》《殷卜辞中所见先公先王续考》。王与罗私交甚好，发现甲骨上很多先公先王的名号与《史记》记载相吻合。印证了《史记》商史的真实性，可以再往前追溯了。建立了二重证据法。
6. 由于民间收藏混乱，中央研究院历史语言研究生开始官方介入，派董作宾到小屯村去印证。方法不科学，听老百姓说哪有就挖哪，收效不大。董作宾，1933发表文字，创建了甲骨的年代学研究。
7. 留学海归的李济、梁思永加入了殷墟发掘，使发掘走向正规化。1928～37，共15次发掘。宫殿、宗庙、王陵、祭祀坑、墓地。
- *甲骨四堂*：罗振玉号雪堂，王国维号观堂，董作宾字彦堂，郭沫若字鼎堂。
- 李济、董作宾后来去了台湾，梁思永1950年患重病，54年去世。
8. 建国后的首次工作：1950，郭宝钧在殷墟发掘武官村大墓。
9. 1958，中科院考古所成立安阳考古队，1959，建立安阳工作站。搞清了殷墟的范围、分布格局，并发现洹北商城。

## 分期与年代
- 盘庚迁殷。之后商人没有再迁过都。《史记》“盘庚迁殷至纣之灭，七百七十三年”，多数学者认为传抄有误，应为273年。夏商周断代工程：253年。
- 分期理论
    1. 邹衡：1956，邹衡根据郑州商城的发现对殷墟提出分三期。根据解放后的新资料分四期。一期：盘庚、小辛、小乙；二期：武丁、祖庚、祖甲；三期：廪辛、康丁、武乙、文丁；四期：帝乙、帝辛。陶器和甲骨刻辞对应。
    2. 社科院安阳工作队：1961，根据大司空村发掘资料分为二期。1962，根据新资料又将各分为早晚期，共四期。后来根据妇好墓、M48判断出武丁在位时间较长，在一期晚段、二期早段。80年代，在三家庄发现早于大司空一期的遗存，又把殷墟文化一期分为早、晚两段。
- 相对年代：晚于二里岗，早于西周
- 绝对年代：前1300～1046。
- 陶器特征
    1. 灰陶为主，由早到晚夹砂灰陶比例下降，泥质灰陶、红陶比例上升。红陶属于三、四期。
    2. 纹饰：绳纹为主，由规整的中细绳纹到凌乱的粗绳纹；三角划纹，殷墟二期偏晚出现，大量流行于三、四期。
    3. 居址与墓葬要分别分期再综合：墓葬中的鬲、簋、豆在居址中也见到，但爵、觚在居址中不易见到。
        1. 鬲变扁方体，裆变低，足根消失。二里岗的足比较外撇，殷墟不撇，变方形 。
        2. 簋腹由倒梯形到半球形，晚期腹部多有三角划纹
        3. 豆颈部变矮
        4. 爵、斝逐渐粗糙，明器化：爵的流退化，觚体变瘦小，拿一小块泥用手捏成的，不实用。
    4. 炊器：空三足器，鬲、甗，与二里岗相似；
    5. 盛贮器：簋、豆、盆、罍、罐。
    6. 酒器：觚、爵
    7. 有少量制作很规整的白陶器，和几何形的印纹硬陶， 仿青铜器制作。二里头也有。

## 范围与布局
- 殷墟保护区：安阳市区西北郊，太行山东麓，华北大平原，洹河穿城而过，30km2。没有发现城。几乎只有王陵区在洹河北岸。
- 洹北商城：1999年发现，在殷墟保护区外东北方，方形。年代总体上早于保护区。
- 殷墟能申遗成功和很早就形成保护区有关。
1. ### 宫殿宗庙区
- 以洹河南岸的小屯村、花园庄为中心
- 未发现城。北、东以洹河河道为屏障，南、西以壕沟为屏障，形成70万平米的封闭性防御空间。
- 30年代，发掘了53座基址，分甲乙丙三组。
- 甲-12复原成了四阿重屋式建筑。
- 乙-7夯土基址中有很多奠基坑，有狗、小孩，在基址前的庭院内有密集的成排分布的祭祀坑，分三组：北组：5个车马坑；中组：80多个坑，大部分为人坑，多数尸首分离，每坑2～13人不等，没有随葬品。可能是“献俘礼”，打胜仗后在宗庙前祭祀祖先，杀一批人，埋一排。
- 高规格墓葬：妇好墓、M54、M17、M18
- 窖穴：储藏粮食、甲骨、玉石器
#### 妇好墓
- 没有被盗。而王陵区的墓都被盗了。
- 土坑竖穴墓。
- 面积22.4平米，深7.5米。
- 棺、椁、腰坑、壁龛
- 殉人16，殉狗6
- 随葬品1928件，6千多枚海贝。200+铜礼器，755玉器。
- 铜器铭文：“妇好”。“妇好”在甲骨刻辞中记载为武丁的妻子之一。是一位能征善战的女武士，出土大量铜兵器，也有铜镜、骨簪等梳妆用具。
- 国宝级的青铜器：三联甗
- 青铜农具：如铜铲。贵族在农祭时要做象征性的劳动。
- 还有很多精美玉器：玉凤、玉人；是贵族女性专用
- 象牙：觚形杯
- 墓口有发现柱洞，有学者认为是墓葬建筑，但是是孤例，现在复原成了一个亭子“享堂”，是对妇好墓进行定期祭祀的场所。后来在山东滕州前掌大商周墓遗址发现明显的柱洞痕迹，证实高规格的商墓有地上建筑。
- 妇好墓、M54为何不在王陵区，而在宫殿区：妇好墓有人牲的遗骨，但没有墓主人的遗骨，有学者提出这是一个衣冠冢，妇好可能是征战中死在外地了，遗骨没有运回，在宫殿区为她举行了盛大的下葬仪式。M54，铭文“亚长”，墓主为长姓高级军事贵族，也直接葬在宫殿区。二人的墓都是殷墟二期的，是在宫殿仍在使用时就葬在宫殿区了。还有小屯的M17、M18也是宫殿区的高级贵族墓葬。
- 妇好33岁去世，可能是难产，也可能是战死或旧伤复发。可能是武丁（在位59年）出于思念将她葬在宫殿前。
2. ### 王陵区
- 以洹河北岸的武官村、侯家庄为中心，也称“西北岗王陵区”。现在通过在墓坑轮廓上种低矮的松柏树标明墓坑所在位置。
- 二里头、二里岗都没有发现王陵区。
- 基本是在解放前发掘完的。全部被盗。解放后发掘了87M1和1576“甲大墓”。
    - 甲大墓有墓坑，没有墓道，也没有下葬仍和东西，推测为纣王墓。纣王突然死亡，墓没有修完，也没有入葬。
- 有4条、2条、1条墓道的大墓。分西区、东区。
- 东区有密集成排的祭祀坑，推算有2500座以上。
1. #### M1001
    - 4条墓道
    - 人牲很多。
    - 大型方形腰坑，殉狗、人（武士）
2. 武官村大墓
    - 郭宝钧发掘
    - 2条墓道
    - 二层台上挖坑，殉人有陪葬青铜器，殉稀有动物：鹿、猴
    - 大量人牲
3. M260
    - 1条墓道
    - 有可能出土了司母戊鼎
    - 棺椁的保存较好
- 被盗情况、原因：巨大的圆形盗坑，内有西周的戈、镞、陶器，不同墓坑的回填土中有能拼成同一件陶器的碎片。表现出是大规模、有组织的同时盗掘。推测为西周初年，周武王死后，商纣王的儿子武庚和管叔、蔡叔叛乱，周公二次东征，平定叛乱后把墓挖了，盗宝、掘祖坟，使其丧失认祖归宗的心理。
- 大规模祭祀坑：完整的人；专门的肢体坑、人头坑；活埋（多为儿童）、腰斩、肢解、解手足。人牲的埋藏情况还不如动物。在早年的学术气氛中，认为是奴隶。近年借助体质人类学鉴定、甲骨文记载，认定为异族战俘。晚商的作战对象是西北方的羌人，将捕获的战俘圈养起来，祭祀的时候使用。
3. ### 族邑居址
- 发掘的比较少。灶在屋角，烟道烟到屋外。
4. ### 手工业作坊
- 铸铜作坊遗址里也有祭祀坑。分层，有上千牛的门齿、下颚
5. ### 族墓地
- 建国后安阳队的主要工作。万座以上。
- 生前聚族而居，死后聚族而葬。
- 一个族墓地可分出若干墓区，一个墓区又可根据随葬情况分出若干墓组，可将人群分级。同一墓区的铜器铭文符号一致，不同墓区的不同。规格相同的聚集在一起。陶器也分区。
- 对墓：异穴并葬、同穴并葬，异性。代表了晚商普通平民的一夫一妻的婚姻状态。
- 有祭祀性遗存：墓葬内的腰坑、人祭；墓地上的人坑、兽坑、人兽合埋坑、车马坑。车马坑分陪葬、祭祀。

## 分布与区域类型
- 空间分布相对二里岗有很大收缩
    - 南边退缩到河南南部，淮河流域也退缩
    - 东边略有推进
    - 西边退到关中东部
    - 北边从桑干河退到拒马河以南
- 变化原因
    - 内因：商王室内部争权夺利，出现九世之乱，内部动荡，对外围控制力减弱
    - 外因：地方土著文化增强，挤压殷墟文化。如河套地区的西岔文化、晋陕高原地区的李家崖文化、扶风刘家文化、荆州周梁玉桥文化、岳阳铜鼓山类型、吴城文化。李家崖和吴城都有城堡。军事色彩浓厚。
- 区域类型
    1. 苏埠屯
    2. 前掌大
    3. 罗山天湖
    4. 老牛坡
    5. 旌介
    6. 定州北庄子
