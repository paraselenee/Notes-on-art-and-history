# 五、长江流域和华南地区新石器时代晚期文化
## 长江中游地区
### 一、大溪文化
- 重庆巫山县大溪遗址。位于长江三峡
- 1925美国探险队曾调查发现石器陶片。五十年代以后正式调查发掘，陆续发现同类遗存，70年代初正式命名。
- 分布
    - 主要在长江中游的两湖（洞庭湖、洪湖）平原地区。西起瞿塘峡东口，东至武汉，南抵洞庭湖，北达荆山和大洪山南麓。地势平坦，河湖密布，气候温暖湿润。
- 重点遗址
    1. 大溪遗址。
    2. 中堡岛遗址，湖北宜昌。现为长江三峡水库大坝址。有大溪一至四期和屈家岭文化遗存
    3. 关庙山遗址，湖北枝江县。保存较好的红烧土房屋，是大溪文化首次发现。大溪-屈家岭-石家河叠压。
    4. 城头山遗址，湖南澧县。大溪文化圆形城址一座，略呈圆形，城垣外圆直径340米，夯土城墙高出地面5～6米，围绕城垣的护城河宽35米，城内面积8万平米。城内有祭坛、房址、制陶作坊、屈肢葬墓、瓮棺葬墓、大溪文化水稻田遗迹。炭化稻谷、数十种植物子粒、竹和芦苇编织物、木桨、船艄等。
    5. 汤家岗遗址，湖南安乡县。精细的印纹白陶。
    6. 三元宫遗址，湖南澧县。成片的大块红烧土建筑遗迹。湖南省首次发现新石器时代墓葬。
- 陶器
    - 红陶为主，有夹炭、夹砂、泥质的红陶，夹蚌或稻壳末的粗泥红陶；夹砂黑陶、泥质的黑陶、灰陶；细腻薄胎橙黄陶；个别白陶。红陶普遍施红陶衣。有的外红里黑。
    - 从早到晚：红陶减少，黑、灰陶增加，有相当数量的泥质磨光黑陶
    - 特征：成组排列的单体戳印纹和浅浮雕式篦印纹。薄胎彩陶。
    - 圜底釜、鼎、豆、圈足盘、小口高领罐、壶、簋形器、圈足碗、单耳杯、曲腹杯、筒形瓶、盆、钵、器盖、器座、支座。
    - 陶器烧成温度较低，硬度不高，但有制作精美和体型很大的。白陶、薄胎彩陶代表了大溪文化制陶工艺的成就。
        - 白陶在前一阶段的早市下层文化中出现。白陶多为圈足盘，胎质细腻，造型规整，通体戳印复杂图案，层次分明，具有浅浮雕的风格。汤家岗类型最发达。
        - 薄胎彩陶常见圈足的杯、碗，个别有单耳。胎厚1～2毫米。有的甚至在器内、器底涂衣施彩。色衣和彩纹浓淡相间，有晕染作风，十分精美，是大溪文化特征。
    - 彩陶：红衣黑彩为主，还有白衣、橙黄衣的红、黑、棕、白彩。钵、罐、豆、壶、筒形器、器座。绞索纹、“人”字纹、漩涡纹、水波纹、网纹、草叶纹、太阳纹、禾穗纹、豆荚纹和各种几何纹饰。受庙底沟影响。
    - 彩绘陶：朱绘于磨光黑陶上为主。斜线、水涡、曲折、平行条纹等，数量很少。
    - 陶塑：汤家岗遗址，陶人面、狗头；大溪遗址，陶猪。
    - 陶响球：空心，内装小石子或泥丸，外表有多个镂孔和刻划纹饰，可能是玩具或原始乐器。（还有实心陶球）
- 石器 骨器
    - 存在较多中大型石斧。
    - 凿。一般为圭形，是大溪文化最有特点的石器。
    - 牙、蚌器很少。
    - 大溪遗址有牙镯、蚌玦。
- 玉器
    - 璜、玦、镯、坠、环、管等，均为小型装饰品。
    - 浅浮雕双面人面纹佩、圆雕双人像。
- 地方类型
    - 意见尚不统一
    1. 关庙山类型
        - 北部
        - 红衣陶为主，流行绳纹，彩陶发达。
        - 石斧是主要特点，圭形凿为其独有
    2. 汤家岗类型
        - 南部
        - 红陶为主，黑陶比例相当大，白陶也多：有浅浮雕风格的白陶圈足盘最有代表性。镂孔发达。
        - 石器不多
- 年代与分期
    - 绝对年代：前4500或4600～3300
    - 分期不统一，暂分三期：质地、陶色、彩绘、纹饰、典型器都在变。
    - 通过叠压关系可知，大溪文化晚于城背溪、彭头山、皂市下层文化，早于屈家岭文化。
    - 来源：不同地区的大溪文化来源不同。城背溪-->关庙山类型，皂市下层-->汤家岗类型。分布范围重合，文化面貌相似，有继承。
    - 去向：发展为屈家岭文化。鄂西地区的大溪文化上层往往都有屈家岭。也可能是不同发源地的并行发展的文化。
    - 影响：受庙底沟影响，尖底瓶、绘圆点弧线三角纹与花斑蚊图案的彩陶；也在豫南、鄂北、汉中的仰韶文化遗址中传入大溪文化因素。玉器种类源于长江下游的河姆渡、崧泽、良渚。
- 聚落与建筑
    1. 修筑房屋前，先平整地坪
    2. 有一定的防雨防潮和加固措施。用火烧烤地面和墙壁，用红烧土铺垫居住面和散水（四面坡）。大溪文化早期已经出现了红烧土做垫层、烧烤墙面与居住面等成熟的建筑技术（如关庙山遗址）。
    3. 大量使用当地盛产的竹作为建筑材料，流行的编竹夹泥再烧烤的建墙方式。
    - 澧县城头山城址，建于大溪文化早期，是中国最早的城址之一。
- 经济生活
    - 几乎所有遗址都有稻作遗存。陶器和红烧土中普遍羼有稻草、稻谷壳。城头山古稻田，三条人工堆砌的田埂组成长方形田丘地，还有与稻田配套的排水沟和蓄水坑组成的灌溉系统，曾经是全世界最早的水稻田遗迹（14年在江苏宿迁（顺山集文化）韩井遗址发现距今八千年的水稻田遗址）
    - 渔猎经济在三峡地区比在两湖平原地区重要。鱼在某些遗址中用于随葬。
- 文化与艺术
    - 刻划符号：有烧前、烧后刻划两种，多为前者。宜昌杨家湾遗址最多。刻在圈足器底外，翻过来才能看见。一般一件器物上刻划一个符号，有固定的意义，可能是记事符号，或是与某种精神活动有关的记事符号，甚至是原始文字。
    - 装饰花纹：戳印、彩画。
    - 雕塑：玉器、陶器
- 墓葬
    - 土坑竖穴单人葬，未见合葬。流行直肢、屈肢葬。极少数附身直肢。
    - 流行下肢弯曲成度很大的屈肢葬，特点鲜明。推测是捆绑后掩埋。
    - 大部分有随葬品，有的是打碎掩埋，一般放在头两侧。
    - 大溪墓底有随葬鱼、狗、龟。
    - 儿童不论瓮棺或土坑葬都单独挖竖穴埋葬，与大人交错葬在同一个公共墓地中。
- 信仰与习俗
    - 城头山遗址祭坛：椭圆形，面积超过200平米，上有圆形祭坑、瓮棺葬、土坑墓等。一随葬品丰富的大型土坑墓，男性，仰身直肢，颈部有2件玉璜，骨架左侧放一儿童头骨，两侧放鼎、圈足盘、豆、盖豆、碗、盖碗等陶器。年代不晚于前4000年，使用至前3800年。祭坛周围还有40多个祭祀坑，有坑内倒扣十余件装满稻米（炭化稻米的草木灰）的陶器。祭祀对象不明
- 社会发展状况
    - 早期较平均，后期出现明显分化。

### 二、屈家岭文化
- 湖北京山屈家岭遗址。发现上千处遗址
- 分布：以江汉平原为中心。北越桐柏山达南阳盆地，影响甚至可达黄河沿岸，西北至武当山，西至西陵峡，南抵洞庭湖南岸，东至黄冈-黄石一带，东北以大别山为界。（包含鄂中、东、西北，豫西南，洞庭湖）
- 重点遗址
    1. 屈家岭遗址，湖北京山市。大面积红烧土建筑。独特的彩陶、彩绘陶、彩陶纺轮。
    2. 放鹰台遗址，湖北武汉武昌区。数十座屈家岭文化墓葬。包括大溪、屈家岭、石家河。
    3. 阴湘城遗址，湖北荆州市荆州区。发现圆角方形的城址，有城垣、壕沟，城墙全长900米。城内文化有大溪、屈家岭、石家河、商周。
    4. 走马岭遗址，湖北石首市。有城垣、壕沟，面积8万平米。城址始建于屈家岭早期，是屈家岭文化重要城址。有大溪、屈家岭、石家河遗迹。
    5. 城头山遗址，湖南澧县。城头山主要为屈家岭遗存，有墓葬500座，居住区400平米，有夯土台基的完整房址2座，一条红烧土铺成、两侧有排水沟的道路。是洞庭湖地区屈家岭文化的代表遗址。
    6. 划城岗遗址，湖南安乡县。大溪、屈家岭、石家河。屈家岭墓葬较多。
    7. 门板湾遗址，湖北应城县。城址、房址。
- 陶器
    - 泥质、夹炭、夹砂。泥质占多数。灰陶较多，黑陶次之，黄、红更少。
    - 部分为轮制：杯、碗壁薄，近似蛋壳陶。
    - 绝大多数为素面。
    - 彩陶：较常见。泥质红陶，施红/白衣，绘黑彩，有浓淡相间的晕染法。极少的朱绘黑陶。彩陶圈足壶、彩陶器盖为屈家岭文化独有。
        - 薄胎彩陶近似蛋壳陶。泥质，反复筛选、淘洗泥料，快轮成形。多为平底小杯，新出现喇叭形小杯。胎厚0.5～2毫米。陶衣有橙红、灰、黑色，多为两、三色兼施或两色重叠，内外兼施。多为黑彩，有晕染作风。图案不如大溪的薄胎彩陶复杂。应该是作礼器。
    - 典型器：双腹器、高圈足器。双腹碗、双腹豆、双腹盆形鼎、簋、高领罐、圈足壶、薄胎彩陶碗、高圈足杯、喇叭形杯。双腹：器身仰折。
    - 彩陶纺轮：橙黄色陶衣，在单面和侧面绘红褐色彩纹。以圆孔为中心，布局匀称。
- 地方类型：
    - 意见不统一
    1. 屈家岭类型。泥质灰、黑陶为主。手制为主，已使用轮制。弦纹和镂孔常见。独有：宽扁足双腹盆形鼎、甑。
    2. 划城岗类型。主要器类和屈家岭类型基本相同，独有：长颈盖壶、尖底尊、小口罐等。
    3. 青龙泉二期类型。郧县青龙泉。基本特点相同，独有：刻槽盆。
- 年代与分期
    - 绝对年代：前3400～2500
    - 分期意见不统一，各个遗址分期数不同
    - 直接渊源是大溪文化，后继是石家河文化。
    - 在偏北的地方叠压在仰韶文化之上，但不是源于仰韶文化。
    - 交流：仰韶文化中心地区、大汶口墓地、庙底沟二期。
- 建筑
    - 应城门板湾遗址
        - 门板湾城：城墙始建于屈家岭文化晚期，400x550。废弃于石家河文化中期。城墙为堆筑而成，未见夯打痕迹（夯窝）。在屈家岭文化晚期之前，以门板湾城壕聚落为中心，已形成了面积为110万平米的聚落群。但后来由于洪水的泛滥，一些居民主动撤离了西部的房屋，修建了面积20万平米的城。
        - F1：叠压在城墙下，可能是当时为了筑城而主动拆除的。F1四开间，110平米，坐南朝北，墙体为土坯垒砌，内外抹细黄泥，墙体最高2.2米。房间一字排开，有门道相通，南墙上有6个方形落地窗，窗框腐朽但仍有印痕，还能看出横梁和边框的榫卯结构。有3个屋内有火塘，最东间无对外的门和火塘，可能是储藏室。北面有内走廊。北、东有院落和围墙，北部院落中还有一屋，似为门房。
    - 淅川黄楝树遗址
        - 排列整齐的单间、两间、连间套房。木骨泥墙，居住面用黄沙土铺垫并夯打，有的抹白灰面。
    - 天门石家河遗址群
        - 石家河城：以城址为中心的聚落群。城址面积120万平米，长江中游地区最大史前城址。墙宽50米，残高6～8米，城壕宽60～100米，深2米，始建于屈家岭文化晚期。
        - 谭家岭F1：居住面下垫有红烧土，表面经火烧烤，南面铺一层竹席
        - 肖家屋脊F15：房基底下对扣两碗作为奠基祭品。
- 经济生活
    - 稻作为主。粳稻为主。
- 习俗
    - 中堡岛遗址：23个器物坑密集分布。坑内物品摆放有序，玉器置于一侧，石器放在一起且方向一致，陶器有高圈足杯、双腹豆、双腹碗等，上下摆放有序。可能是祭祀坑。
- 社会发展状况
    - 从建筑可以看出一个大家族分成几个小家庭
    - 陶祖的出现表现出对男性的崇拜
    - 墓葬有贫富贵贱之分
    - 以城址为中心的聚落发展起来，出现了中心聚落和普通聚落的二级结构
    - 以上，显示出氏族社会内部发生了深刻巨大的变化，为长江中游文明的起源打下了坚实基础。

### 三、雕龙碑三期类文化遗存
- 湖北枣阳雕龙碑遗址
- 有较多屈家岭文化因素，又有仰韶文化因素，还有自身新的文化特征。
- 新生产工具：陶研磨棒、陶多孔器。
- 绝对年代：前3300～2800
- 猪下颌骨大量随葬，最多的一个墓随葬72副，说明养猪业极其发达。
- 2座7间房的地面建筑，超过100平米。红烧土奠基、砌墙。居住面是修建在厚厚的红烧土基层上，铺垫着单层或多层石灰，涂抹得平整光亮，再经火烧烤，有的变成光亮的青绿色，像水泥地一样。有木质门框和推拉式屋门结构的痕迹，是史前房屋建筑考古的重要发现。

## 长江三角洲地区
### 一、河姆渡文化
- 浙江宁波余姚河姆渡遗址2、3、4层遗存
- 分布：宁绍平原、舟山群岛。
- 陶器
    - 釜为主要炊器。
    - 早期：夹炭黑陶为主，夹砂黑陶次之。一些器物上有动植物图案。各种釜，流行口部和腰部出脊。方钵。划纹常见：短斜线组成的条带和多角星纹，叶脉纹，锯齿纹，重圈纹。
    - 中期：同早期，并出现少量泥质红陶。釜的种类减少，脊退化成凸棱。出现腹部加牛鼻耳的罐。钵敛口有鋬。盆有宽折沿。有灶。
    - 晚期：夹砂灰陶为主，新出现泥质灰陶、黑衣陶、夹砂红陶。流行多边口沿釜。受马家浜文化影响出现腰沿釜。新出现鼎和喇叭状高柄豆。有盉。支座作兽头状。动植物图案不见。
    - 泥条盘筑，晚期出现慢轮修整。
- 骨、角、牙、木（漆）器
    - 早期在遗物中比例相当高。晚期数量骤减。
    - 骨耜、笄、镞、哨
    - 牙雕蝶形器
    - 木器：带榫卯的干栏建筑构建、桨、耜、纺轮、筒、刀、蝶形器、锯形器。种类繁多，在国内考古学文化中独树一帜。特殊的埋藏环境利于木器保存。
        - 有硬木制作的织机零件。
        - 朱漆木碗：整块木料剖挖而成。外壁为瓜棱形，带圈足。器表髹以朱红色漆，漆皮多有剥落。是新石器时代最早的漆器标本之一。说明朱色漆的调制已经达到了一定的技术水准。
        - 缠藤篾朱漆木筒：系采用整段圆木刳空制成，形似一段中空的竹筒。内、外壁磋磨光洁，断面近似椭圆，器壁厚薄均匀，外壁两端缠有多道藤箍。出土时外壁有髹漆，呈金黄色。因保存条件不足，现已褪色。
- 石、玉器：数量较少。环、玦、珠、璜。
- 年代与分期
    - 绝对年代：前5000～4000，与马家浜文化年代相当。可能还要更早。
    - 分三期，对应马家浜文化的一、二&三、四期。
    - 来源未知。晚于城背溪、彭头山、皂市下层、跨湖桥，应该有更古老的源头。
    - 余姚井头山贝丘遗址，前6300～5800，近两年刚刚发现。
    - 河姆渡与同期的马家浜文化关系密切。器类大致相同，器物形态和纹饰也相似。
    - 该地区下一时段的文化有崧泽文化和其他文化。
    - 貌似没有明显的继承和发展关系？
- 经济生活
    - 相当成熟的稻作农业。普遍有稻谷遗存。籼。骨耜是重要的农具，用大型哺乳动物的肩胛骨制成，大量出土。
    - 大量出土骨镞、骨哨，用于诱捕野兽。
    - 船桨，柳叶形，说明已能操桨行舟捞鱼。
- 房屋建筑
    - 发现桩木和散落的建筑构建，有在埋桩木的坑底垫木板，或直接打入生土层。主要有带榫卯的方、圆木，企口板。
    - 推测为全木结构的干栏建筑
        - 干栏建筑：以木柱和连接其间的木梁为骨架，木梁上铺垫地板，形成高离地面的建筑基础，再在此基础上架设房屋梁柱，以木、竹席等围墙盖顶。木构件之间以榫卯结合，地板间以企口拼接。
    - 大房子，有前廊。排桩保存完好。房内可能用席子之类分隔成若干间。
    - 有木构水井遗迹，上有井亭。
- 文化与艺术
    - 双鸟朝阳纹蝶形牙雕，反映了丰富的精神文化生活。还有陶器上刻画的动植物图案、陶塑。
- 墓葬
    - 土坑竖穴、偏东向，单人。河姆渡多为侧身屈肢，塔山多为仰身直肢。
    - 随葬日用陶器、工具等，数量均少，无差异。
    - 社会分化不明显。

### 二、马家浜文化
- 浙江嘉兴马家浜遗址。70s末确立为马家浜文化。
- 桐乡罗家角遗址证明了河姆渡和马家浜文化是相互影响的并存关系。
- 分布
    - 古长江以南，古钱塘江以北，茅山和天目山以东：苏南和江北的太湖周围地区。
    1. 马家浜遗址，浙江嘉兴。建筑遗迹、人骨。
    2. 罗家角遗址，浙江桐乡。遗物最丰富的马家浜文化遗址。有榫卯的建筑构建，大量兽骨。
    3. 圩墩遗址，江苏常州。大量墓葬、随葬品，木桨、橹
- 陶器
    - 夹砂、夹蚌、夹炭、泥质。以夹砂红陶，外红里黑、表红胎黑的泥质红陶为主。
    - 均为手制。
    - 特点：带腰沿和带鋬的釜、带把杯、三足或平底盉、多边形宽沿盘、鼎、喇叭状把豆。
- 年代与分期
    - 绝对年代：前5000～4000
    - 分四期。
- 经济生活
    - 前5500～4500，海平面上升，太湖周围水位上涨，只有高亢地带适合人居。其后，海面较为稳定，许多地区因沉积作用而加高，海岸线逐渐后退，虽然也会受到特大高潮的威胁，但已初步具备人类栖息条件。
    - 苏州草鞋山遗址：发现了马家浜文化晚期的稻田遗迹。由成排的3～5平米的小浅坑组成，一侧有井，另一侧有一个大水塘。部分田块有睡口与水塘相连。土层中有大量水稻硅酸体。粳比籼多。粳、籼谷粒块不混合，说明人们可能已经认识到两个品种不同。
    - 粳、籼都有，籼的比例略大，这是早期稻作农业的特点。稻作农业相当发达。
    - 农具只有骨耜，用于翻土、培筑田埂
