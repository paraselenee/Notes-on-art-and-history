# 抗日战争专题 北京大学 王奇生
- 站在胜利者的角度会看战争，会弱化它的残酷性和困难程度。要从当时人所处的境遇去出发。
- 后人会带着后知后觉的心态把问题简单化，觉得很多事情理所当然。也不会注意到当时的大背景。
- 毛泽东批判过两种声音：亡国论、速胜论。大多数私密性材料(日记、书信)认为抗战必败。
- 陈寅恪的判断是：毫无胜利的可能性，不如把华北也给日本，剩下的地方可以三五百年之后再复兴，而如果抗战到底，有可能彻底亡国。但是当时的公开材料则不能表达这种声音。
- 南京沦陷后，撤到武汉，国民政府行政院的高官日记中记录，同事们天天吃喝嫖赌。而后来的历史记录里都是慷慨激昂的。抗战胜利以后，他自我检讨，人生从来没有在武汉那几个月那么堕落过，那是亡国前的所作所为。
- 在举国绝望的背景下，再来看蒋介石、毛泽东。
- 蒋介石的日记不能说完全没有的作秀的成分，但是坚持抗战的立场基本没有动摇过。高官纷纷让他认清现实，低调认罪，但他没有动摇。他的基本考虑是：
    1. 如果是两国直接的战争，中国赢不了，但这场战争一定会发展成国际性战争，只要成为国际性战争，中国就有可能赢得胜利。有这种判断的人在当时也不多。他认为日本和苏联一定会开战，三年之内、两年之内、一年、半年、三个月，上帝保佑打起来吧。1905年日俄战争之后，日本胜，从此之后日本把俄国、苏联当作战略假想敌。日本陆军认为占领东北、华北是都是进军苏联的前奏。所以蒋介石的战略是“苦撑待变”。结果日本没有跟苏联打，索性跟美国打了。日本海军主张打美国。这也跟斯大林的狡猾有关，他为了极力避免两线作战，稳住东方，跟日本签订条约，又极力鼓动中共发动抗日，从1935开始，共产国际七大，研究的就是如何把战火维持在中日之间，尽力维持中日的战争。共产国际实际上是被莫斯科，即斯大林控制的，是带有苏联立场的。
    2. 蒋介石对日本非常了解：野心巨大，欲望无穷，求和解决不了问题。他明白中日必有一战，中国只能希望尽量希望战争晚一点，短一点，但不可避免。
    3. 从中共关系考虑，假如国民党向日本求和，那么外战一止，内战必起。中共会趁机借民族主义的旗号推翻国民党。所以国民党与其亡于内战，不如亡于外战，至少留个好名声。
- 国民党的体制主要是蒋介石个人独裁，而在这个问题上恰恰是独裁起了作用。他是很孤立的。
- 相比之下，汪精卫的投敌，是因为他认为走和平路线会在党内获得支持，但谁都不敢出来求和，因为历史上求和的人从来没有好下场。他到了越南河内后，在香港《南华日报》上公开发表通电，还给当时的行政院院长发了一个个人电报，说的大意是：大家都不下地狱，只有我来下地狱了。如果只是想做汉奸，没有必要发通电。当然也不可否认汪精卫与蒋介石有个人的政治斗争，但是也是因为他清楚国民党内的求和声音很强。
- 中共党内也有对战争的悲观论调，但毛泽东不悲观，他也认为中国有胜利的希望。毛泽东的判断：
    - 从两国的基本国情，而不是两国的实力来考虑：中国是广土众民，日本是小国寡民。日本最多只能占领大城市和交通要道，所以广大的农村正是中国的抗战复兴基地。而当时大部分人看不到这一点，都是从实力来看。
- 《论持久战》：即使在今天，整个看一看关于抗战的各种言论和论著，可以发现几乎没有第二个人对这场战争站在战略的高度，做了如此深入、透彻的分析，国共之内、之外都没有。他也参考了很多东西。让周恩来他们在国统区收集军事资料，结果全是战术方面的。但他不关心战术，关心战略，重新搜集，收获了克劳塞维茨的《战争论》，如获至宝，在延安找几个人，翻译一章，讨论一章。
- 《论持久战》发表以后，对中共党内产生了非常重要的影响，也确立了毛在党内的领袖地位。遵义会议时只是让他再度进入高层，但总书记还是张闻天，军事最高决策是周恩来，只是在长征中发现周的军事指挥能力不如毛泽东，就慢慢让权给毛。到1939以后，张才慢慢把总书记的权利让给毛，毛在名义上还不是。中共非常重视理论，崇拜共产国际，毛虽然有军事指挥才能，但很多人还是瞧不起他的。
- 中共是外来政党。
    - 《党史》的前三章，中共产生的北背景：欧洲列强一战，无暇登陆，给中共的民族工业很好发展时机；工人阶级壮大；中共成立。然而一战并没有使民族工业发展，因为欧洲在打仗，海路断了，机器运不到中国，中国刚刚起步，也不会造机器，没办法办厂。只是当时的报纸有“抓住机会”的声音，但实际上并没有抓住。
    - 世界各国共产党几乎都是在1920～1926建立的。背景几乎都是来自共产国际。1919年3月2日列宁成立共产国际。(狭义的“五四”是学生运动，广义的是新文化运动，三四十年代后慢慢合成了“五四新文化运动”。学生运动没多大影响，新文化运动举起了“民主”“科学”。)还是“三二”的影响更大。列宁的计划非常宏大：1.建立世界共产党 2.发动一场世界革命 3.革命成功后建立世界苏维埃共和国。
    - 共产国际成立后，莫斯科派人到各国去帮助建党，除了欧洲的一些是从社会民主党分化出的共产党，其他各国都是新建的。建立之后要加入共产国际，作为共产国际的支部。共产国际是高度集权、纪律严格的。一切行动都要听从莫斯科的指导，完全没有自主性。了解这个才能明白毛泽东的独立自主是多么的不容易，为什么一定要反教条主义。陈独秀曾试图跟莫斯科讨价还价，也没有成功，所以后来说他们的“右边机会主义”这些事实上都是莫斯科的指导。莫斯科的指令“正确的要执行，错误的要先执行再说，有意见再反应”。1928年，有60+国家加入了共产党，都在莫斯科的统一指挥下搞世界革命。在当时的交通、通讯的条件下难以想象。商业电报费太贵，只在紧急情况下用，一般情况下写信。上海到莫斯科一来一回至少两三个月。而莫斯科对于中国的情况知之甚少，政策经常脱离中国国情，而且还跟不上形势，产生错误在所难免。
    - 1931年左右，莫斯科给上海送来一台电台，可以直通电报了，不用走商业电报了，特别方便。于是产生了新的问题，对中共的控制更加厉害。三天两头就来一个几千上万字的电报，独立自主的空间非常小。
    - “1927年大革命(国民革命)失败，发动南昌起义，然后秋收暴动，在土地革命战争中开辟农村包围城市的道路”，其实不是这么回事。一直到1933，上海中央被迫迁往中央苏区之前，中共的革命基本上还是走城市暴动的道路。毛泽东走向井冈山，其实都不是中共主流的暴动，农村暴动还是旁门左道。城市暴动路线也来自莫斯科。法国大革命、十月革命、辛亥革命、武昌起义都是城市暴动。暴动成功就是革命，失败就是叛乱。而毛的道路在世界革命史上没有先例。
    - 以往在城市暴动时期也不用建立职业革命武装，把党建好，把群众组织好，等革命时机来了，一暴动就胜利了。而大革命的失败恰恰是因为中共没有职业革命武装，而蒋介石有。蒋之所以有是来自于孙中山的革命总结。孙中山当年革命时也没有建立职业革命武装，多次失败后，意识到必须要建立职业革命武装。联俄以后，孙中山首先提出要莫斯科帮助他建立军队，莫斯科坚决反对，孙一再坚持，莫斯科最终同意帮他建立黄埔军校。所以后来蒋才能以军队对付共产党。
- 中共早期的崛起是很顺利的。
    - 1927年国共分家之前，中共有近6万党员，4万团员，960万农会会员，390万工会会员，15万童子团。这个规模在当时的共产党里仅次于苏联，大大出乎共产国际的意料。共产国际最早把希望寄托在欧洲，没想到中共发展得如此快，把中共树为典范。
    - 没有搞土地革命也能有960万农会会员，说明农民能否动员跟搞不搞土地革命没有必然关系。甚至在解放战争时期，中共最终的胜利跟土改也没有直接的关系。过去常说农民在革命胜利后能分到土地，所以都积极当兵。现在看也不是事实。1947年，中共的兵源来自于土改翻身农民的只有20%多，其余来自国军俘虏。1948年，近90%来自国军俘虏。开始的时候还给他们几天的政治思想教育，到了1948年，上午抓来俘虏，换身衣服，改名“解放战士”，下午就上战场。所以后来毛承认“拿国民党的兵、国民党的枪打败了国民党”。1949以后，土改的作用才显现出来，中国增收农业税的能力大大提升。之前国民党根本收不上来农业税，而土改彻底摸清了土地占有情况。
    - 莫斯科在陈独秀之后任命的领袖：瞿秋白、张闻天、秦邦宪、王明都在莫斯科呆过、留学过。只有陈独秀是例外。中共早期能顺利崛起和陈独秀也有直接关系。多数国家的共产党在早期就夭折了，最后建立政权的更是极少数。像中共这样基本靠自己打下来的很少。很多国家共产党在早期有内部派系斗争问题。国民党内部也有严重的派系斗争，而中共早期没有派系。过去认为是信仰的力量，但是其他国家的共产党内派系也很多，原因很可能是党魁。国产国际在其他国家建党时找的多是年轻人，因为中年人不愿意革命了，都是年轻的话，谁做党魁都不服，又谁都想做，于是就三五成群，都想得到莫斯科的承认。而中共找到了40岁的陈独秀来做党魁。1920年，莫斯科到北大来找人，因为北大老师里有个俄国教授，可以借他的关系来入手。首先介绍了28岁的胡适，可是胡适晚上有应酬，胡适就介绍了老乡陈独秀。
    - 胡适太温和。文学革命最早是他打起来的，给陈独秀的《新青年》写了一篇《文学改良刍议》，而被陈独秀改名为《文学革命》，于是就发动了文学革命。陈独秀说“我不管是写文章还是说话，我会说极正确的话，我也会说极错误的话，但我绝对不说那种不痛不痒的话”。革命家就是要走极端，一定要与众不同。“九·一三”之后，清理林彪的笔记，写道：一号(毛)有个毛病，凡事他征求你的意见，只要你先说，他一定要说得跟你不一样，所以他征求你的意见时，你一定要听他先说，然后你附和他就可以了。
    - 陈独秀通过新文化运动之后，已经是全国知识界的领袖人物。毛这批五四青年都是读陈独秀的文章长大的，无比信仰陈独秀，绝对不会挑战他的权威。后来才批评陈有“家长之作风”。当时只有李大钊有可能另立一派，但他是一个纯粹的马克思主义者，没有这个念头。三大中心：李大钊领导北方，陈独秀在上海，陈延年(陈独秀长子)在广州。
- 毛泽东的崛起
    - 1933年上海中央的人搬到苏区后和毛不和，周恩来等人集体给莫斯科写信告状，说他太有个性了，搞不到一起，所以把他挂起来了，晾在一边。
    - 毛的机会：长征前夕，中共和莫斯科通电报的密码搞丢了，毛泽东自立一党，在长征路上拉拢张闻天、王稼祥形成一个三人小集团，趁机召开遵义会议，颠覆中央。遵义会议就类似一个政变，把莫斯科指派的领导给颠覆掉了。
    - 1936年8月，莫斯科派人把密码带到陕北，重新恢复联络的时候，毛已经掌控大局，莫斯科也只能承认他了。
    - 1935年，共产国际七大，莫斯科检讨，认识到高度集权、铁的纪律导致了很多问题，开始逐渐放权，增强各国的独立自主性。所以也没有继续接管中共的事情。
    - 1943年，共产国际解散。1945，中共七大、延安整风才能顺利召开。延安整风有几个层次：
        1. 整周恩来、张闻天、王明等中共高层，他们都听从共产国际的路线，要整他们的教条主义。张闻天是总书记，所以还是要从王明下手。毛不仅写了《论持久战》，还写了《矛盾论》《实践论》等哲学文章，因为他要树立党内信仰，让那些搞理论的人相信他，不然毛要从党内平辈的人中出头太难了。
        - 孙中山也不是革命党内的绝对领袖，直到民国建立后，都没有树立独一无二的地位。1914年孙中山建立中华革命党，总结革命失败的原因就是党内很多人不听他的话，所要重新把国民党改名为中华革命党，且要求党员要按手印，向孙氏效忠，于是党内很多人没有加入中华革命党。此时党内还有另一个党魁，黄兴，死于1916，但是死后黄兴的支持者也没有转而支持孙中山。于是在1918、1919年孙中山写了《孙文学说》，树立个人崇拜。(《毛泽东思想》是第二个，生前用自己的名字命名主义、思想的。第三个是现在的。)《孙文学说》的“知难行易”讲的是政治斗争。党内支持黄兴的人就是因为黄有很强的执行力，而孙中山空谈理论，外号“孙大炮”，“铁路计划”直到今天都没有实现。孙的自我定位是，我这样的人在中国要五百年才会产生一个，而中国的问题在我的时候还不解决，未来就还要过五百年。但党内并不认同他。“知难行易”就是要说“知”是最难的，别不把我当回事。1924国民党一大召开，启用一批中青年党员，把老的那批晾在一边，才终于树立权威。
        - 中共七大的会议记录中可以看到：周恩来的讲话，一上来就把毛吹捧一番。每个人上来都要先拍他马屁。张闻天写了九万字的反省笔记。证明了七大时毛已完全控制局面。
        2. 整知识青年。1938年前后，近万的国统区知识青年奔赴延安，后来被称为三八干部。也是延安整风的对象。因为来自国统区，有自由主义，小资产阶级风气。三八干部对中共革命有很重要的作用。1931～1933，苏区普遍搞了一场“肃清反革命”(肃反)运动，简单说就是抓国民党特务。后来证明几乎全是冤假错案，没有几个真正的特务，而他们都被杀了，数字很难统计。1985～1986，平反了很多，共十几万，江西六万多，湖北三万多，河南、湖南各一万，还有安徽、福建、陕西、四川。五十多年后，要提供人证物证，还能平反十多万，可以说肃反全面地整肃掉了党内有知识、有文化的干部。徐向前说，当年在大别山鄂豫皖根据地，被自己杀的人比跟国民党打仗牺牲的人还多。所以1934年长征的时候，官兵基本都是文盲，到了陕北根据地后，三八干部恰好弥补了党内干部和知识分子的真空。
- 长征
    - “王明的左倾机会主义，造成第五次反围剿的失败，导致红军被迫长征”。但是其实即使没有国民党的围剿，苏区也很难长期呆下去。因为苏区的财政就是“打土豪分田地”，杀鸡取卵，没有自己的财政、经济来源。两三年就把一个地方的土豪打干净了，农民又要当红军，妇女儿童还要做后勤，时间长就穷了。只能换地方。所以也有自愿撤离的成分。
    - 如果1937年抗战爆发的时候中共还在大别山、井冈山，届时日本从北方或者上海打，国民党撤往西南，长征北上恰好为中共后来发展提供了非常好的地理空间，又避免了跟国民党的正面冲突。张国焘与毛泽东会师后，张主张南下四川西昌。西南的自然条件比西北要强很多，而西北是穷山恶水。但是毛之所以选西北，是为了跟苏联取得联络，将武器、经济各方面的希望都寄托在苏联上，革命靠自己很难搞下去。北上后本来计划派西路军打下新疆，当时新疆的领导跟苏联走得很近，也想过走外蒙古，当时也在苏联手上。总之北上不是为了建立根据地，到了那里正好发现刘志丹还有一个小根据地，加上打通苏联通道的路线失败，才建立的。
    - 毛泽东到陕北后意识到陕北无法养兵，所以只能往东，跨过黄河进山西。当时的山西是阎锡山的地盘，阎锡山是辛亥革命后的“山西王”，特别有地盘意识，决不允许任何势力进入山西。1936，中共东征，打了阎锡山，起到了非常好的效果。他的手下徐永昌日记中详细记录了阎锡山跟中共交战后，说从未遇到过这么强悍的军队，还预判未来打日本只能寄托在中共军队身上了，未来中国可能是共产党的天下。阎于是怕中共，学中共。日本从华北打进来后，山西首当其冲，阎锡山认为只能靠中共的力量抵抗日本，主动欢迎中共进山西。抗战初期国共合作开拓华北，主要就是开拓山西。整个抗战时期，毛泽东反复跟中共高层交代，要好好跟阎锡山处好关系，一定不能逼迫他，使他倒向日本或国民党中央，使阎锡山在我们和国民党、我们和日本之间起到缓冲地带。1944之前，毛都是尽量迁就。
    - 七七事变：日本学术界的基本看法是：就是一个偶然性事件，不是有预谋、有计划地打响全面战争的开端。材料上来看的确如此，只是后来升级了。北支那驻屯军确实没想到要借这个跟中国开启全面性战争，当时是想像“满洲国”一样建立“华北国”。关东军在建立满洲国后在日本国内受到一致的拥护，出尽风头，所以华北军也想这么做。全面侵华源于日本的陆海军之间竞争。甲午战争是海军出风头，后来陆军打下东北，华北，海军为了政绩，在上海挑起战争。在当时，毛对日本这样分散战斗力感到莫名其妙。蒋纬国认为这是他父亲有意的设计：国民党在上海抵抗了三个月，后来又把华北日军南调到华东地区支援，使日军的进攻路线从自北而南改为自东向西，而江南水乡不利于日本机械化部队，加大了进攻难度。其实也并非蒋的设计。
        - 日本在整个二战期间没有最高军事统帅。美国罗斯福、英国丘吉尔、苏联斯大林、中国蒋介石、纳粹希特勒，就日本没有。陆军海军之上只有天皇。天皇是神，神不能犯错，不愿承担责任。所以日军败在体制。
