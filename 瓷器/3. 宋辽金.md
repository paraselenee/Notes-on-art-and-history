# 三、宋辽金时期瓷器
## 宋词发展面貌概论
- 170多个县发现窑址，宋代占了130多个。可见宋代社会经济繁荣。
- 官民分立，雅俗并进：形成了官窑+几大民窑的窑系。

1. 五大名窑：汝、官、定、钧、哥
    - 汝：河南宝丰清凉寺。80s发现。
    - 官：北宋汴京，南宋修内司、郊坛下。北宋开封在黄泛后被埋；南宋
    - 定：河北曲阳。
    - 钧：河南禹州
    - 哥：龙青/杭州，有争论，主要是传世品
2. 六大窑系：以名窑为中心，窑厂扩大，周围相继仿烧的体系。在销售地又引起仿烧，扩展到他处。各窑系都产很多品种，但也有一个代表性的知名品种
    - 定窑系
    - 磁州窑系
    - 耀州窑系
    - 钧窑系
    - 龙泉窑系
    - 景德镇青白瓷窑系
## 北方地区的主要生产传统
1. 汝窑
    - 河南平顶山市宝丰县清凉寺。
    - 北宋元丰(1078～1085)～宋金交战。南宋初已十分珍贵。现在存世品不足70件。
    - 汝窑既有官窑也有民窑。
    - 胎：细腻致密，香灰色，口沿和底足透过釉处微微带些粉色
    - 釉：浑厚均匀，釉面油润，釉色素洁高雅，青中闪蓝，故有“雨过天青”之美誉。通体施釉。据记载以玛瑙为釉料。玛瑙的主要成分是SiO2，又含铁等着色元素。汝州产玛瑙，宫廷不计成本。有细小的“蟹爪纹”开片。釉中气泡很少。有黑色的缩釉点。
    - 器形：碟、碗、樽(仿汉器)、三足盘。不多。
    - 装饰：基本为素面，有简单的弦纹。
    - 烧制：裹足支烧或裹足刮釉，仅在外底有3或5个细小的“芝麻挣钉”
    - 明清仿烧：明代仿汝釉瓷仅见于景德镇珠山宣德官窑。明代仿宋官窑器一般只注重仿宋器的釉色，造型和款识都显示出宣德官窑瓷器的特征。雍正、乾隆时期以唐英为督陶官的景德镇御窑厂曾大量仿制宋代五大名窑器物，仿汝釉瓷器主要是仿釉色，造型袭古又有创新，器物外底多有以青花书写或以铁锥刻划的本朝篆体年款。
2. 钧窑
    - 河南许昌市禹州市
    - 北宋晚期～元。
    - 胎：厚重，酱色，口沿底足露胎处可以看到黄铜色胎体。元代钧窑器形较大，壁更厚。
    - 釉：上很厚的乳浊釉(乳光釉)，清足，在足底部露胎处刷上一层芝麻酱釉。表面有“蚯蚓走泥纹”“兔丝纹”“泪痕纹”，布满细小的棕眼(缩釉点)。
        - 乳光釉：天然分为两层，在连续玻璃质上层悬浮着另一种液滴状物质，不能熔融的物质还产生小气泡，它们引起光线的散射出白、红、蓝光，产生“乳浊”和“乳光蓝”
        - 民窑器物形制多为碗、盘、瓶、罐之类生活用品，釉色以纯色居多，无官钧之玫瑰紫和海棠红色(官钧也有纯色的)。
        - 纯蓝色的釉从浅到深称为月白、天青、天蓝。在乳光釉中加氧化铁和微量的磷，可能烧出带蓝光的天蓝釉，或者是类似汝窑的颜色。
        - 玫瑰紫釉：含氧化铜，烧出的釉色青中带红。如闪烁在釉色里的青空和云霞。
    - 器形：花盆、花奁。花盆有葵花、海棠、仰钟、菱花、六方、长方等。传世官钧窑的外底大都有从“一”到“十”的数目字刻釉，根据现存实物证明，数目字越小，器物尺寸越大。
    - 烧制：先在八九百度素烧，再重复上釉。不全是裹足支烧，
    - 钧窑系：北宋时影响河南，金、元时期影响到河北、山西。各窑的釉色、器形略有差别。
        - 窑变釉：是清雍正时期仿钧窑瓷器时所繁衍出来的一个新品种，采用两次或多次上釉的方法烧成。所施釉料中除铜外，还含有铁、锰、钴、钛等微量着色金属元素。在烧成过程中，这些金属的氧化物呈色后相互交融，使釉面呈现出缤纷的色彩，其中釉色偏红的称“火焰红”、偏蓝的称“火焰青”。
3. 耀州窑
    - 唐～元，盛于北宋中期。明清仍有烧造，但不在黄堡镇，搬到陈炉镇了。
    - 代表：青釉剔花
    - 品种：青、黑、白、酱、青黄。
    - 胎：香灰色
    - 釉：青、橄榄青色，滋润，玻璃质感强，满釉；酱釉，仿漆器，产量仅次于青瓷。金代釉色青中泛黄。
    - 装饰：题材丰富、活泼流畅、刀锋犀利、用斜刀的剔刻。早期模仿越窑，北宋中期形成自己的风格。题材有牡丹、莲瓣、菊花、婴戏、海水鸭纹。也有瓷塑，药王塑像，孙思邈的家乡也在铜川。
    - 销路：民用为主，也有进贡的。
    - 耀州窑系：宋金时期河南临汝窑、内乡窑、新安窑，广西永福窑(桂林南侧)都烧类似的青瓷。临汝窑仿得很像，但刻花不如耀州窑犀利，仿定窑的印花还不错。
4. 定窑
    - 晚唐～元。
    - 代表：白釉印花
    - 胎：轻薄细腻
    - 釉：白为主，属于暖白色，微闪黄，温润恬静的美感。流釉有“泪痕”。兼烧褐、黑、绿、白釉剔花、酱釉。
    - 器形：生活、供奉、瓷塑
    - 装饰：金、银、锡、铜釦，刻花、划花、印花、浮雕、金彩。线条优雅流畅。也受定州缂丝工艺和金银器纹样影响。
    - 款识：十五种，多与宫廷有关，带“官”字的最多。
    - 装烧
        1. 支圈挂烧：北宋出现，也叫支圈正烧，口沿外撇，胎体普遍较薄，内壁施满釉，外壁在口沿处因与支圈接触形成一周约1厘米宽的无釉涩胎痕。
        2. 支圈覆烧：北宋中期出现，可以充分利用窑的容积，大大提高产量。胎薄，圈足矮，口朝下烧，有芒口，镶釦成“金装定器”。
        3. 涩圈叠烧：金出现，碗内有一圈无釉，产量高，用于烧粗瓷
    - 定窑系：山西平定窑、阳城窑、孟县窑、介休窑，四川彭县窑，耀州窑、磁州窑也仿定窑
        - 介休窑：白釉与定窑很相似。宋初～清。也有黑釉和白釉釉下褐彩。盏的盏心和圈足各3个支钉，是介休窑独创。
5. 磁州窑
    - 邯郸峰峰矿区、磁县观台镇和彭城镇
    - 北宋～元，繁荣于金
    - 代表：白地黑花
    - 胎：粗糙，黄白或浅灰褐色，含杂质，厚重，普遍施化妆土
    - 釉：白、黑、绿、青黄；化妆土剔花的铅釉陶
    - 装饰技法
        1. 白釉划花：线条流畅。叠烧，里面有5个支钉痕。
        2. 白釉剔花：将纹饰以外的地子剔去，使纹饰具有浮雕感，多在器外，缠枝花卉。
        3. 白釉黑彩：最主要的手法。在坯胎上先施一层洁白的化妆土，然后用毛笔蘸“斑花石”彩料描绘纹饰，最后施透明釉，入窑经高温一次烧成。装饰器外、枕面。也有白釉褐/红绿彩，绿釉黑/褐彩。牡丹、荷花、鱼、菊花、凤、诗文、卷草纹、狮子、婴戏，线条都很饱满。
        4. 白釉黑彩划花：高档品。施洁白的化妆土，细黑料绘纹样，在黑色纹样上划画花瓣叶筋，露出白色化妆土，施一层薄而透明的玻璃釉，烧。黑白两色对比强烈。花卉、龙纹。
        - 3和4都叫“白地黑花”
    - 款识：戳印，磁枕戳印张家作坊标识的很多，是有商标性质的作坊款式。
    - 窑：馒头窑，匣钵装烧，三角形支钉，北宋中期开始烧煤
    - 磁州窑系：对河南、陕西、山东、江西、福建都有影响。河南当阳峪窑、集窑、扒村窑，山东淄博窑，山西介休窑，江西吉州窑，福建茶洋窑、磁灶窑。
## 南方地区典型瓷窑与瓷器
- 龙泉窑和景德镇崛起，形成青、白两大窑系。
- 杭州设立修内司、郊坛下两座官窑。
- 福建建窑、德化窑，江西吉州窑相继步入繁荣。
1. 官窑
    - 内府制样，民匠造器，士兵供役。官建，官管，官用。次品就地打碎掩埋。
    - 又称“内窑”。用于宫廷和国家重大仪式典礼。
    - 修内司：高宗南渡后设立的，仿汴京官窑，澄泥作坯，釉色晶莹透澈，粉青，釉有深浅，有蟹爪纹开片“金丝铁线”和“紫口铁足”。开片较浅淡的是“金丝”，较深的是“铁线”。口部釉薄，底足无釉。南方胎土含铁量高，胎呈黑褐色。
    - 郊坛下：南宋初期设立的第二座官窑。胎黑灰色、黑褐色，薄胎厚釉，釉色有粉青、炒米黄等。器形仿商周秦汉青铜器、玉器者甚多，受徽宗复古风气影响，作宫廷陈设用。
    - 宋代官窑瓷器与汝、哥、钧窑瓷器一样，特别注重釉色美。其凝厚的釉层系采取先素烧坯体、然后多次施釉的方法完成的，釉层厚度往往大于坯体的厚度。烧成过程中，选用优质木材作燃料，以保证窑内的温度和气氛。官窑瓷器多为素面，既无精美的雕饰，又无艳彩描绘，多用凸凹直棱或弦纹为饰，在釉面自然优美的片纹点缀下颇显高贵典雅。
    - 开片：釉面开片本因釉与胎的收缩率不一致，冷却时形成一种釉裂胎不裂的现象，古代工匠巧妙地利用错落有致的开片，顺其自然，形成一种妙趣天成的装饰釉。这种装饰主要出现在宋代官窑、哥窑、汝窑青瓷表面。或称“开片”、“龟裂”。
2. 哥窑
    - 尚未发现窑址，主要是传世品。有认为龙泉窑中黑胎的就是哥窑，也是官窑。
    - 代表：金丝铁线
    - 釉：灰、米色，厚润，黑色、米色开片，俗称金丝铁线。开片是比官窑更细碎的“鱼子纹”、满布器体。
    - 胎：黑
    - 烧造：6支钉
    - 造型：玲珑小巧。
3. 龙泉窑
    - 北宋～明中期。南宋、元鼎盛
    - 分布：浙江省丽水市龙泉市金村、大窑、溪口等地，地处浙江南部靠近福建。
    - 龙泉窑系：浙江各地，福建松溪窑、南安窑，江西景德镇窑、吉安永和窑。
    - 三个阶段：
        1. 北宋～南宋前期：薄胎薄釉，石灰釉，淡青、青黄，釉面光莹。日用、冥器。碗、盘、执壶、五管瓶。划花、刻花。
        2. 南宋中期～龙泉窑兴盛期：薄胎厚釉，灰白浅胎、灰黑深胎，石灰碱釉，粉青、梅子青，釉面柔和滋润。日用、文房、鸟盏、佛前供器、香炉、八仙塑像。贴塑、堆塑。仿商周青铜器的鬲式炉、仿周代玉器的琮式瓶和仿汉代铜壶等器物最为精致，风格秀美。
            - 石灰碱釉：高温下粘度较大，不易流釉，可以施厚。在厚釉失透的情况下，北宋龙泉青瓷盛行的刻、划花装饰已不适用，取而代之的是堆塑和浮雕技法。
        3. 兴盛期之后～明中期。虽然地区扩大且产量巨增，但已无法烧出南宋时期的梅子青色。为弥补青瓷呈色的不足，元代龙泉窑青瓷常采用模印、贴塑、镂雕等工艺手段作装饰，以追求图案美和造型美。明代龙泉窑的生产仍很兴旺，成为景德镇以外较大的窑场。明初的龙泉窑瓷器在制作工艺上与元代基本一致，装饰技法以刻、划花为主。胎体厚重，造型雄浑粗犷，在明代瓷器中别具风格。其釉层肥厚，色调虽不如宋代粉青和梅子青那样青翠，却也保持了龙泉窑釉色淡青泛灰、明艳沉着的特色。
    - 釉：犹如青玉，润而不透。烧成温度高，光泽强，玻璃质感强。
    - 胎：加入紫金土(含铁)，降低胎的白度，显得釉色更深沉。
4. 黑瓷生产
    - 黑釉茶盏：福建建阳水吉窑、江西吉州窑
    - 宋代喝茶：把半发酵的膏饼碾成细末，放在茶碗内，冲沸水，水面浮起一层白沫。黑釉茶盏衬托白色茶沫最适宜斗茶，因而受到斗茶者的喜爱。
    1. 建窑
        - 南平市建阳区水吉镇
        - 晚唐五代～元末
        - 代表：黑釉，盏、碗
        - 胎：厚、深、粗。
        - 釉：外壁施釉不到底，垂流严重，有“兔毫”“油滴”“曜变”等结晶纹样
            - 曜变：油滴斑一周蓝色，绝品
            - 兔毫：有金黄色和银白色，最著名
            - 油滴(鹧鸪斑)：一个个金黄或蓝白的小圆点
        - 器形：敛口、敞口，形如漏斗
        - 款识：部分上贡的碗底阴刻或模印“供御”“进琖”
